# Bank Management System (C++)

A robust and professional console-based Bank Management System built using **Advanced C++** and **Object-Oriented Programming (OOP)** principles. 

## Features
The system provides a comprehensive suite of banking functionalities, including:
1.  **Client Management**: Show, Add, Delete, Update, and Find clients.
2.  **Transactions**: Deposit, Withdraw, Total Balances, and Transfer Log.
3.  **User Management**: Full control over system users and permissions.
4.  **Currency Exchange**: Real-time currency conversion and rate management.
5.  **Security**: Secure Login/Register system with encrypted-style file handling.

## 🛠️ Technical Skills Demonstrated
*   **OOP Pillars**: Heavy use of Inheritance (from `clsScreen`), Encapsulation, and Abstraction.
*   **File Handling**: Permanent data storage using `.txt` files.
*   **Architecture**: Clean code structure with a clear separation between logic (Header files) and presentation.
*   **Validation**: Robust input validation system (`clsInputValidate`).
